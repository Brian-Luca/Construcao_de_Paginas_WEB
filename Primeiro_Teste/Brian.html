Cascading Style Sheets

O que é CSS?

CSS é uma linguagem para estilos que define o layout de documentos HTML. Por exemplo, com o CSS pode-se controlar fontes, cores, margens, linhas, alturas, larguras, imagens de fundo,posicionamentos e muito mais...

Qual a diferença entre HTML e CSS?

HTML é uma linguagem básica para estruturar os conteúdos que definem uma página web. CSS é o código que se vai utilizar para formatar os conteúdos que foram estruturados na criação da página web.
