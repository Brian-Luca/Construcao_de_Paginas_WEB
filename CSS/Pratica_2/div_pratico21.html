Blog

O que é?

Milhares de novos blogs são criados todos os dias por pessoas que querem partilhar os seus pensamentos e o seu modo de vida on-line. Mas quantos blogs realmente fazem a diferença? Criar um blog é muito fácil hoje em dia. Não importa se é um adolescente ou uma pessoa de idade, apenas é necessário ter um computador, acesso à internet e utilizar uma plataforma para contruir o blog.

Por onde começar

Existem muitas plataformas para iniciar um blog. Blogger, Tumblr, WordPress.com e muitas outras. Se a ideia é um blog profissional, é altamente recomendável usar o WordPress como plataforma, mas isso requer a compra de um nome de domínio, hospedagem na web e instalação do WordPress num servidor web.
Começar um blog é fácil, mas manter um blog é muito difícil e demora tempo para conseguir aumentar as visualizações.
