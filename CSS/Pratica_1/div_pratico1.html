Corpo

Layout

A forma mais eficaz de criar páginas com bons layouts é através da aplicação de blocos utilizando a tag <div> e em seguida modificando-os através da utilização do CSS. Com CSS pode-se alterar a altura, largura, cor, posição e outros elementos tanto das div's como de todos os outros elementos html.

Desenvolvimento Prático

Para aprendermos a desenvolver um layout com div e css, vamos desenvolver um exemplo completo para aplicar as técnicas necessárias à concretização do objetivo.

O layout que vamos desenvolver é composto por um cabeçalho, uma menu horizontal, uma coluna de conteúdo principal, uma barra lateral à direita, e um rodapé. A página será centrada horizontalmente na janela de menuegador, como mostra a imagem a seguir.
